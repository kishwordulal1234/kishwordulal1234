## How to Set Up the Snake Animation

To set up the snake animation contribution graph shown in the template, you'll need to:

1. Create a new repository named `yourusername` (same as your GitHub username)
2. Create a `.github/workflows` directory in that repository
3. Add a YAML file for the GitHub Action that generates the snake animation

Here's the workflow file you should create:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *" # Runs every 12 hours
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: Platane/snk@master
        id: snake-gif
        with:
          github_user_name: yourusername
          svg_out_path: dist/github-contribution-grid-snake.svg
          
      - uses: crazy-max/ghaction-github-pages@v2.1.3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
